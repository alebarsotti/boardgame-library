# Archived Specs

This folder stores specs that have already been implemented or are no longer part of the active roadmap.

Rules for archived specs:

- keep the original numbered filename
- move the file here after implementation is complete
- keep links relative when referencing archived specs from active docs
- avoid editing archived specs unless the goal is to record implementation outcome or clarify history

The active roadmap should remain in the parent `docs/` directory so current priorities are easy to scan.
